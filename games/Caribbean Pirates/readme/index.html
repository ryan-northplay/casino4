A) Description - top

The Caribbean Pirates is a HTML5 casino game.
This game has been developed in HTML5 / JavaScript and third-party library phaser - http://phaser.io/

Features:
• Audio / Music on off
• Fullscreen
• Multilanguage can choose in the settings (English / German / Spanish)
• Autoplay
• Freegames
• 13 Winlines
• All symbols animated
• Jackpot

The ZIP package contains the game with 1920 x 1080 resolution that scales to fit the current screen device.
The game is fully compatible with all most common mobile devices.

C) CSS Files - top

We using two CSS files in this Game.The first one is a generic reset file. Many browser interpret the default behavior of html elements differently. By using a general reset CSS file, we can work round this. This file also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.

The second file contains the background color of the page and maybe for future updates.

E) JavaScript - top

This game contains five JavaScript files.

  1. jQuery - lib/jquery.min.js
  2. jStorage v0.4.12 - lib/jstorage.js
  3. Phaser v2.10 - lib/phaser.min.js
  4. Configuration File - src/config.js
  5. Our custom script - src/game.js
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. jStorage is a cross-browser key-value store database to store data locally in the browser - jStorage supports all major browsers, both in desktop (yes - even Internet Explorer 6) and in mobile.
  3. Phaser is an open source HTML5 game framework created by Photon Storm. It's designed to create games that will run on desktop and mobile web browsers.
  4. In The configuration file you can change debugmode (so developers get a console output), bets, jackpot, chances and many more.
